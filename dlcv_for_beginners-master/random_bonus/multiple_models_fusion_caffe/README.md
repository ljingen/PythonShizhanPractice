## 生成融合后的prototxt

> python rename_n_freeze_layers.py input_model output_model prefix

拷贝从data层之后开始到最后要进行融合的特征层，比如ip1位置的prototxt，放到一个prototxt中，然后在开始加上data层，融合层可以用concat进行拼接，然后再接fc或是其他操作。
